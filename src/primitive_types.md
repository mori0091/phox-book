# Primitive Types

| type   | values                | description                                    |
|--------|-----------------------|------------------------------------------------|
| `()`   | `()`                  | Unit type, that consists of single value `()`. |
| `Bool` | `true`, `fales`       | Boolean type. True or false.                   |
| `Int`  | `0`, `100`, `-1`, ... | Integer type. (currently 64-bit integer)       |
